# AI Knowledge & Model Management Frontend

This is a Next.js + TypeScript frontend using Shadcn UI for managing datasets, models, and AI studio features.

## Requirements

- Node.js 18+
- Yarn or npm

## Setup

```bash
npm install
# or
yarn install
```

## Development

```bash
npm run dev
# or
yarn dev
```

## Build

```bash
npm run build
# or
yarn build
```

## Start (Production)

```bash
npm start
# or
yarn start
```

## Lint

```bash
npm run lint
# or
yarn lint
```

## Generate OpenAPI Types

```bash
npm run typegen
# or
yarn typegen
```

## Features

- Dashboard landing page
- Datasets: upload files, create datasets, manage knowledge bases
- Models: manage local, provider, and custom models, add API keys, fine-tune models
- AI Studio: advanced AI operations

## Tech Stack

- Next.js (latest)
- React (latest)
- TypeScript
- Shadcn UI
- Axios
- OpenAPI types
